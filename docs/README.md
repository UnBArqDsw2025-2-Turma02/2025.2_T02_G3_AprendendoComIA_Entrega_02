## Informações Complementares 

Para executar a documentação em ambiente local siga os seguintes passos:

### Instale o docsify

Execute o comando:

```shell
npm i docsify-cli -g
```

### Executando localmente

Para iniciar o site localmente, utilize o comando:

```shell
docsify serve ./docs
```
